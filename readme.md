# Инструкция по сборке и запуску
Проект использует `Makefile` для автоматизации настройки окружения и запуска.

*   **Подготовка (venv + зависимости):**
    ```bash
    make
    ```
*   **Запуск обучения и оценки:**
    ```bash
    make run
    ```
*   **Очистка проекта (удаление venv и кэша):**
    ```bash
    make clean
    ```

    ```bash
    make help
    ```

1. Цель работы
Реализовать алгоритм обучения нейронной сети с нуля, используя исключительно библиотеку NumPy (без применения готовых фреймворков глубокого обучения, таких как PyTorch или TensorFlow).

В рамках данной работы была решена расширенная задача: вместо базового бинарного классификатора на синтетических данных реализован многослойный перцептрон (MLP) для классификации реальных изображений одежды из набора данных Fashion-MNIST (10 классов, размерность признакового пространства — 784).

2. Архитектура решения
Для обеспечения читаемости, модульности и масштабируемости программного кода применена объектно-ориентированная архитектура. Базовым элементом является абстрактный интерфейс Layer, от которого наследуются функциональные слои сети.

2.1. Полносвязный слой (Dense)

Слой выполняет линейную трансформацию Z=XW+b.

Инициализация: Для предотвращения проблемы затухания и взрыва градиентов использована инициализация Каймана Хе (He Initialization). Масштаб начальных весов вычисляется как 2.0/input_size.

Регуляризация (Дополнительное задание): В метод обратного прохода (backward) встроена L2-регуляризация (Weight Decay). Осуществляется математическое вычисление градиента со штрафом за сложность модели путем прибавления к базовому градиенту значения весов, умноженного на коэффициент λ:
dW=dW+λW

Оптимизатор (Дополнительное задание): Реализован алгоритм градиентного спуска с моментом (Momentum SGD) и его модификация — импульс Нестерова (Nesterov Accelerated Gradient). Слой инкапсулирует вектор скорости v, что позволяет сглаживать колебания градиента и ускорять сходимость к оптимуму.

Формула стандартного момента:
v = βv + (1 - β)dW
W = W - ηv

Формула Нестерова (в реализованной эффективной форме):
v_new = βv - ηdW
W = W - βv + (1 + β)v_new
v = v_new

2.2. Слой нелинейности (ReLU)

Выполняет функцию активации f(x)=max(0,x). Обратный проход реализован вычислительно эффективно с помощью применения векторной булевой маски.

2.3. Слой потерь (SoftmaxCrossEntropy)

Вычисление функции Softmax и перекрестной энтропии (Cross-Entropy) объединено в единый слой для повышения вычислительной стабильности.
Для предотвращения переполнения при вычислении экспонент применен метод вычитания максимального логита (Log-Sum-Exp trick).

2.4. Интеграция конвейера (MLP)

Класс инкапсулирует упорядоченный список слоев. Реализует методы forward, backward и fit (Mini-batch GD с перемешиванием).

4. Экспериментальное исследование гиперпараметров
Для проведения экспериментов разработан скрипт `experiments.py`.

4.1. Влияние скорости обучения (Learning Rate)
Исследуемые значения: 0.001, 0.01, 0.1
Результат: η=0.1 обеспечивает наиболее быструю сходимость. η=0.01 дает стабильный результат. η=0.001 слишком мал для 10 эпох.

Исследуемые значения: 16, 64, 256

4.3. Влияние инициализации весов
Исследуемые подходы: Zero, He, Large (N(0,10)).
Результат: Zero-init приводит к симметрии (Loss ~2.3). Large-init дестабилизирует сеть. He-init показывает лучший результат.

4.4. Влияние импульса Нестерова (Nesterov Momentum)
Исследуемые подходы: Standard Momentum vs Nesterov Momentum.
Результат: Импульс Нестерова позволяет "заглядывать вперед", что обеспечивает более стабильную сходимость и меньшее количество осцилляций вблизи минимума по сравнению со стандартным моментом.
